# Weaven Shop

Welcome to Weaven! A modern, full-stack E-commerce web application built for premium clothing and accessories.

## Features

- **Dynamic Frontend:** Built with React, Vite, React Router, Tailwind CSS, and DaisyUI.
- **Engaging UI:** Includes a dynamic hero banner powered by Swiper and Framer Motion for smooth animations.
- **Product Management:** Complete Product Details page featuring image galleries, size/quantity selectors, and live stock tracking.
- **Authentication:** Integrated Firebase Authentication for secure user Login and Sign Up.
- **Express Backend:** Powered by Node.js and Express, securely storing product catalogs in MongoDB using Mongoose.
- **Admin Dashboard:** A dedicated, protected `/admin` route for store owners to view statistics, manage "All Products", and "Add Products" directly to the database.
- **Direct Contact Integration:** Social integration allowing users to contact the seller directly via WhatsApp and Facebook Messenger from the Product Details page.

## Tech Stack

- **Frontend:** React 19, Vite, Tailwind CSS v4, DaisyUI, Framer Motion, Swiper, React Hook Form, Axios.
- **Backend:** Node.js, Express, Mongoose, CORS, Dotenv.
- **Database:** MongoDB
- **Authentication:** Firebase Auth
